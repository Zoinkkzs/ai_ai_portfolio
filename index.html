--- START OF FILE README.md --- # Thanh Huy Portfolio An interactive portfolio built with React, Three.js, and Tailwind CSS. This project uses an `index.html` structure with embedded Babel scripts to ensure full compatibility with static hosting environments that might not have a build step for TSX files. ## Features - **3D Molecular Viewer**: Explore chemical structures like Benzene and Glucopyranose in 3D. - **Eclipse Simulation**: Visualize solar eclipses with dynamic orbital mechanics. - **Black Hole Hero Section**: Canvas-based interactive background. ## Development - All application logic resides in `index.html`. - `App.tsx` and `index.tsx` are deprecated placeholders.
